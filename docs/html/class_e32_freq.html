Detailed Description

E32 Radio Frequency Setting.

E32 - 'Channel' corresponds to 00H to 1FH assigns 410 to 441MHz Each (1) Per 1-MHz adjust [1F=31]. Default is 17H(23DEC) + 410 = 433. Default Parameters: C0.00.00.1A.17.44 C AH.AL.SP.CH.OPTION Current Parameters: .00.00.1A.17.C7 ( Then set address to : 1234 )
