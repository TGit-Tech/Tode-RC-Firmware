Detailed Description

LIST of LISTS Top-Level Navigator of multiple MenuList (s).

Navigation is preformed by sending keypad pressed to the Top-Level Navigator. The key will undergo a round-robin calling sequence to List-Navigate then Item-Navigate objects. If the object has SEL = Item-Element 'NavSelected' the key is used by said object and a returning command-type non-keypad key is returned. See KEY
