\doxysubsection{Detailed Description}
{\bfseries{L\+I\+ST of L\+I\+S\+TS}} Top-\/\+Level \mbox{\hyperlink{class_navigator}{Navigator}} of multiple \mbox{\hyperlink{class_menu_list}{Menu\+List}} (s). 

Navigation is preformed by sending keypad pressed to the Top-\/\+Level \mbox{\hyperlink{class_navigator}{Navigator}}. The key will undergo a round-\/robin calling sequence to List-\/\+Navigate then Item-\/\+Navigate objects. If the object has \mbox{\hyperlink{group___s_e_l}{S\+EL = Item-\/\+Element \textquotesingle{}Nav\+Selected\textquotesingle{}}} the key is used by said object and a returning {\itshape command-\/type} non-\/keypad key is returned.
